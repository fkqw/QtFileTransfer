QtFileTransfer
==============

## Introduction
QtFileTransfer 是一款方便、簡單、快速的跨平台檔案傳輸工具.  

## Features
* 支援檔案拖曳至視窗 (Client)  
* 跨平台檔案傳輸 (Windows and Linux)  
* 顯示傳輸進度

## Windows Release 佈署
Qt_5_10_0_MSVC2015_32bit 編譯出來的執行檔是 32-bit,  
但放在 64-bit 的 windows 的作業系統要配上 64-bit 的 msvcp140.dll 和 64-bit 的 vcruntime140.dll.
